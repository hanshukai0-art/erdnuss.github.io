====================================
图片文件夹说明
====================================

📷 这个文件夹用于存放你的摄影作品

请将你的照片文件放在这里！


📐 图片建议规格：
--------------
• 推荐尺寸：宽度 1200-2400px
• 推荐比例：4:5（竖版），例如 1600x2000px
• 文件格式：JPG
• 文件大小：建议每张 200KB - 800KB
• 色彩模式：RGB


📝 命名建议：
-----------
城市主题：
  city-1.jpg
  city-2.jpg
  city-3.jpg

人文主题：
  humanity-1.jpg
  humanity-2.jpg
  humanity-3.jpg

自然主题：
  nature-1.jpg
  nature-2.jpg
  nature-3.jpg


⚠️ 重要提醒：
-----------
1. 文件名要和 script.js 中配置的完全一致
2. 注意大小写（city-1.jpg ≠ City-1.jpg）
3. 不要使用中文文件名
4. 不要使用空格，用横线 "-" 代替


🎨 图片优化：
-----------
上传前建议使用以下工具压缩图片：

• TinyPNG：https://tinypng.com
• Squoosh：https://squoosh.app
• ImageOptim（Mac）


压缩可以：
✓ 减小文件大小 50-70%
✓ 保持视觉质量
✓ 加快网站加载速度


📂 当前文件夹状态：
----------------
如果看到示例图片（city-1.jpg 等），
那些是占位图片，可以删除并替换为你自己的作品。


✨ 准备好了吗？
-------------
将你的精彩作品放进来，
然后在 script.js 中配置信息，
你的作品集就可以展示了！
